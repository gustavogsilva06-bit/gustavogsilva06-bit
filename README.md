## Projeto em destaque

### Bigode Pay

Fintech SaaS premium para operacoes de checkout e pagamento.

Principais recursos:

- checkout com Mercado Pago;
- dashboard operacional;
- planos e assinaturas;
- comunidade para assinantes;
- webhooks e integracoes;
- Bigode AI;
- deploy em Cloudflare Workers.
